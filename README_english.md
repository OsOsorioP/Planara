# Planara

**Planara** is a web application designed to efficiently manage task lists. It offers features for both standard users and administrators, enabling easy customization and supervision of tasks.  
<p align="center">
  <img src="https://github.com/OsOsorioP/Planara/blob/main/Imagenes/Planara01.webp" alt="home" title="Home Screen" width="660">
</p>

## Features

- **Task management:** Create, edit, and delete tasks intuitively.  
- **Admin profile:** Advanced features to manage users and monitor tasks.  
- **Modern interface:** Built with modern technologies for a smooth and fast experience.  
- **Robust backend:** Secure and scalable API to handle data.  

## Technologies Used

- **Frontend:**
  - [React](https://reactjs.org/) + [Vite](https://vitejs.dev/)
  - Styles: CSS + [Bootstrap](https://getbootstrap.com/)
- **Backend:**
  - [Express.js](https://expressjs.com/)
  - Database: [MongoDB](https://www.mongodb.com/)
- **Language:** [TypeScript](https://www.typescriptlang.org/)
